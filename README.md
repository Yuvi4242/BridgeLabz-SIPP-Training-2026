# BridgeLabz Java Practice Workspace

This repository contains Java practice projects organized by topic and difficulty level.

## Project structure

- `core-java-practice`
  - `gcr-codebase`
    - `ControlFlow`
      - `L1_Practice_Problem`
      - `L2_Practice_Problem`
      - `L3_Practice_Problem`
    - `practice-problem`
    - `ProgrammingElements`
      - `L1_Practice_Problem`
      - `L2_Practice_Problem`
    - `leetcode-codebase`.

## Contents

- `ControlFlow` folders contain structured practice programs for levels 1, 2, and 3.
- `ProgrammingElements` folders contain beginner Java exercises and small practice programs.
- `leetcode-codebase` contains solved LeetCode-style problems.
- `practice-problem` contains basic Java examples from earlier practice sets.

## Notes

- All Java files are standalone and compile independently.
- `Scanner sc` is used for user input in generated practice files.
- No helper methods are used in the generated practice programs.

## How to run

Open a file in the desired folder and run it with Java SE 23 or your installed JDK.

```bash
javac YourFileName.java
java YourFileName
```
